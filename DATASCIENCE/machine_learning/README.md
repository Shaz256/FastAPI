# Customer Purchase Prediction

End-to-end machine learning pipeline to predict whether a customer will make a purchase.

## What this project demonstrates

- Data loading & validation
- Train/test split
- Logistic Regression modeling
- Model evaluation (Accuracy + ROC AUC)
- Basic EDA

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn

## How to run

```bash
cd models
python train_model.py
